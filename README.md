# 어디로든🧳

#### 여행지 정보 공유 및 예약 안드로이드 앱

<주요 기능>
- 지역별 관광지,음식점, 축제 등 정보 조회
- 지도로 한눈에 확인 하는 여행지 관련 정보 
- 사용자끼리 정보를 공유하고 이야기를 나눌 수 있는 커뮤니티
- 여행 상품을 예약
